Running the script will query IIS for all websites that’s physical path ends with ‘content’. This indicates that the website was powerdeployed.


I RECOMMEND running this in Preview Mode first and manually inspecting the list of folders that are going to be deleted. If happy, then run the script in delete mode, see below.


Preview Mode - Shows what folders it thinks should be deleted

>CleanupIISWebsiteDeployments.ps1 


Delete Mode - Deletes folders it thinks should be delete

>CleanupIISWebsiteDeployments.ps1 -delete true
